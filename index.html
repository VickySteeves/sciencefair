Why ScienceFair?

How we access, read and reuse scientific literature is largely controlled by a few vast publishing organisations. This is holding society back.

Many wonderful innovations are being explored outside those organisations, but they are rarely integrated into the platforms where people actually access science.

We have a vision of a different, better, future for science. A future that's more fair, inclusive and open. A future that makes it easy to innovate and explore and where users control and customise their experience.

ScienceFair aims to help pave the road to that future. The main thing that sets it apart? Freedom from centralised control.

Our version 1 release is just the first step. We hope you'll try it and join us.

Thanks, The ScienceFair team

Features

  • Search, collect, read and analyse scientific papers.
  • Clean, modern user interface.
  • Rich interactive reading experience.
  • Distributed data store.
  • Control the datasources you use.
  • Text-mining ready corpus.

Getting Started

When you download and install ScienceFair, you'll be able to search the eLife journal (more sources to come soon!), download articles locally, and read and tag them. Here's how that looks!

1. Search & Discovery

ScienceFair provides an instant search of your local collection and remote sources. Search on whatever terms you'd like!

2. Exploring the Lit

ScienceFair has an in-app reader, putting key aspects of articles such as references and figures into the sidebar for convenience.

3. Tagging

ScienceFair gives users the freedom to visualize metadata, as well as tag materials with their own keywords for quicker, easier access.

Technology

  • Science-focused JATS XML reader provided by lens.
  • Distributed data sharing using Dat's hyperdrive.
  • Fast local search index built on searchindex.
  • Created using web technologies with choo and electron.
